Atari XL/XE

My very first computer was an Atari 400, where I learned how to write basic programs and began using a computer regularly.

In this section I'll discuss the tools I use and the programs I've written using modern computers that run on the Atari 8bit line. There is a great community out there who continues to write games and utilities for the older machines, of which I consider myself a proud member.
